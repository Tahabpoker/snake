# Snake Game 

A small snake game made with Python and pygame.

Honestly made this at 3 in the morning because I couldn't sleep

## Requirements

- Python 3
- pygame

## Install

Using uv:

```bash
uv sync
```

Or with pip:

```bash
pip install pygame
```

## Run

```bash
python main.py
```

or

```bash
uv run python main.py
```

## Controls

- Arrow keys or WASD to move

eat food = grow
eat self = end
